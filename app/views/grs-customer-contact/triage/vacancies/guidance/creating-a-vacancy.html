Stage 1: Advert Only vacancy request

Step 1

An ‘Advert Only’ vacancy should be created by the Vacancy Holder (VH) on a ‘Vacancy Holder’ profile. This will be received through the Dashboard and it will go into the ‘Vacancies awaiting recruitment review’ as normal.

Note: If you’re not sure whether the vacancy is ‘Advert Only’, you can check the ‘History’ to see if the VH has created it using an ‘Advert Only’ template.

All communications for Advert Only vacancies will need to go through CRM at present as there is no facility to push back on vX.
