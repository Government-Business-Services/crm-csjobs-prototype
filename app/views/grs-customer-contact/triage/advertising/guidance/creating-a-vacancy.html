Stage 1: Advert Only vacancy request

Step 1

An ‘Advert Only’ vacancy should be created by the Vacancy Holder (VH) on a ‘Vacancy Holder’ profile. This will be received through the Dashboard and it will go into the ‘Vacancies awaiting recruitment review’ as normal.

Note: If you’re not sure whether the vacancy is ‘Advert Only’, you can check the ‘History’ to see if the VH has created it using an ‘Advert Only’ template.

All communications for Advert Only vacancies will need to go through CRM at present as there is no facility to push back on vX.

Stage 2: Locating Vacancy on VX

Step 1

Enter the vacancy reference number into the ‘Search Vacancies’ box and click the search icon.

If ‘Search Vacancies’ is not shown, you can change the search by using the small dropdown arrow.

Step 2

You will be taken to the vacancy ‘Summary’ tab.

Step 3

Note: You must be on a ‘Recruiter’ profile to ‘Edit’ an Advert Only vacancy.

Stage 3: Approval

Step 1

Go to the Advert Guidance Hub, where you will see ‘Stage 2: Approval’ on your department's advert guidance, to check what approval to advertise is required.

Step 2

Select the ‘Vacancy Forms’ tab and see the ‘Approval’ section to check the correct approval has been received.

Stage 4: Vacancy details form - Recruiter Actions

Step 1

Ensure the information regarding the application process is clear for candidates in the advert.

Make sure the GRS Recruitment email address ‘recruitment.grs@cabinetoffice.gov.uk’ is not included in the advert. If it is, replace this email address with the one provided by the Vacancy Holder for applications to be sent to, or for candidate queries.

Note: No standard wording will be required in the advert, unless including this has been agreed with the department.

Step 2

You will need to complete all mandatory fields (*) before you are able to save the vacancy. Once all mandatory fields have been completed then select the 'Submit' button.

If you select ‘Submit’ before completing all mandatory fields, the page will refresh and all incomplete mandatory fields will be highlighted in red; you will need to complete any fields highlighted red before continuing.

Questions displayed to candidates are indicated by the following warning: (Field visible to candidate) *

Step 3

You will be taken back to the vacancy ‘Summary’ tab.

Stage 5: Manually activate vacancy

Step 1

You will need to manually activate the vacancy before it can be posted to Civil Service Jobs.

Step 2

Select ‘Edit’ on the vacancy ‘Summary’ tab, then tick the ‘Is active’ box.

Step 3

Check that the ‘Preferred Go Live Date’ and the ‘Preferred Vacancy Closing Date’ are correct.

Note (Future Go Live Date): The ‘preferred Go Live date’ needs to be a past or present day to post a vacancy with a future ‘Go Live’ date. For example, if today’s date is 1 July and you want to complete the ‘external postings tab’ with a go live date of 8 July, the preferred go live date must be today's date or earlier. Amend the dates if required.

Step 4

Select ‘Save’.

Stage 6: Posting the vacancy

Step 1

Select the ‘External Postings’ tab, then select the ‘Add’ Button.

Step 2

Within the ‘Destination’ dropdown list select ‘Post to Civil Service Jobs’ and select ‘Next’.

Step 3

Enter the ‘Live Date’ and ‘Posting Closing Date’.

Note: The ‘Posting Closing Date’ must be the same as the ‘Preferred Vacancy Closing Date’ that you entered on the ‘Summary’ tab. This will prevent candidates being able to apply after the closing date.

Step 4

Select ‘Next’. You will be taken to the ‘New Posting’ page.

Step 5

Select ‘Next’. This will activate your vacancy.

Step 6

Select the ‘Confirm vacancy posted’ button to change the vacancy status; an email will be sent to the Vacancy Holder automatically to advise that the vacancy has been posted.

Stage 7: Create vacancy on CRM

Step 1

Login to CRM.

Step 2

Select ‘+ Create Vacancy’ from the drop down on the ‘Vacancies’ tab.

Step 3

‘Is this a WCN (V9) Vacancy?’ - Select ‘No’ for VX vacancies so that CRM automatically hides the fields that are no longer required.

Step 4

Input the ‘CSJobs No.’ and complete all of the required fields.

Step 5

Select ‘Save’.

Step 6

Save the advert request email to the vacancy.

Stage 8: Issue email to VH - CRM

Step 1

Check the ‘CRM Action’ section of your department's advert guidance for any special instructions.

Step 1

Send an ‘Advert Live’ email to the VH.

Check the email is linked to the vacancy:

  • Select the blue ‘Email’ button from the top right-hand corner; this will open an email to the VH.
  • Complete the ‘From’ and ‘Subject’ fields.
  • Select ‘Templates’ and input VX into the search field.
  • Select ‘VX - To VH - Vacancy Posted (Advert Only)’.
  • Check and complete any required information on the template.
  • Select ‘Send’.
