To request an amendment to a vacancy’s advertisement, you need to provide information about the vacancy. Make sure you have all the relevant details ready, such as the vacancy’s reference number.

This is for correcting errors or making changes to:

  • the vacancy’s job title
  • the vacancy holder’s details
  • the vacancy’s department
  • location and salary details

Before you start
